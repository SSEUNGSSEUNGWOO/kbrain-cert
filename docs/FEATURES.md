# kbrain-cert — 기능 리스트

역할별로 분류. **[핵심]** = MVP 필수. **[확장]** = M4 이후 추가. **⚠️** = 원본 plan 이슈 반영 지점.

---

## 역할 (Role)

| 역할 | 설명 | Supabase Auth 구분 |
|---|---|---|
| **admin** | 문제·응시자·시험·결과 전체 관리 | `role='admin'` claim |
| **applicant** | 시험 응시, 결과 확인 | `role='applicant'` |
| **grader** | 서술형·작업형 수동 채점 | `role='grader'` (admin이 겸임 가능) |

## Admin (관리자) 기능

### 문제은행 관리 [핵심]
- 문제 CRUD (객관식 · 단답 · 서술형 · 작업형)
- 문제 유형별 필드
  - 객관식: `choices[]`, `correct_answer_index`
  - 단답: `correct_answer` (문자열, 여러 개 허용)
  - 서술형: `rubric` (채점 기준)
  - 작업형(생성형AI 등): `submission_slots[]` (파일 업로드/URL/텍스트), `rubric`
- ⚠️ **정답·rubric 필드는 서버 컴포넌트에서만 렌더**, 클라 API 응답에는 절대 미포함 (이슈 #2 근본 해결)
- ⚠️ **정답 입력 칸 옆 "🔒 응시자 비공개" 배지**, placeholder 입력 칸 위 "⚠ 응시자에게 그대로 노출됨" 캡션 (이슈 #2 UI 가드)

### 문제 세트 관리 [핵심]
- 세트 CRUD, 세트에 문제 추가/순서 조정
- JSON 파일 일괄 업로드
  - ⚠️ **업로드 파서에서 `submission_slots[].placeholder == correct_answer`면 경고 토스트** (이슈 #2)
- ⚠️ **세트별 `proctoring_disabled` 토글** (이슈 #1)
  - 라벨: "부정행위 감지 비활성화 (생성형AI 작업형 등 외부 도구 허용 구간)"
  - 도움말: "이 세트의 문제 구간에는 전체화면 이탈·얼굴·음성 감지가 일시 중지됩니다"

### 시험 관리 [핵심]
- 시험 정의: 세트 조합 · 시간 · 응시 대상 · 절대시간/상대시간 모드
- 응시자 배정 (개별/CSV 일괄)
- 응시 링크 발급 · 접속 코드 생성

### 응시 모니터링 [핵심]
- 실시간 응시 현황 (진행/제출/이탈)
- 응시자별 감독 이벤트 로그 (얼굴 미검출 · 다인원 · 시선 이탈 · 음성 감지 · 전체화면 이탈)
- 개별 응시자 강제 종료 · 시간 연장

### 채점 관리 [핵심]
- 자동채점 실행 (객관식 · 단답)
- 수동채점 대기열 · 채점자 배정
- ⚠️ **점수 표시는 (raw/max)*100 통일** (이슈 #3)
- 부분점수 · 감점 규칙 편집

### 결과·통계 [핵심]
- 응시자별 결과 페이지 · CSV export
- ⚠️ **CSV 헤더 · CrossTable 라벨 모두 `/100` 표기** (이슈 #3)
- 합격/불합격 판정 (환산 기준)
- 전체 통계 (평균 · 표준편차 · 문항별 정답률)

### 시스템 [확장]
- 감사 로그 (누가 언제 문제/점수 변경)
- 계정·역할 관리
- 도메인 브랜딩 (로고 · 색상)

## Applicant (응시자) 기능

### 시험 대기실 [핵심]
- 로그인 · 응시 링크/코드로 진입
- 시험 안내 · 준비 체크리스트 (카메라 · 마이크 · 브라우저 권한)
- 감독 사전 캘리브레이션 (얼굴 등록 등)

### 시험 응시 [핵심]
- 문제 네비게이션 (이전/다음/번호 점프)
- **타이머**
  - 서버 시간 기반, 최초 접속 시 동기화
  - ⚠️ `start_time` 누락 시 **3회 재시도 (토스트 안내) 후 자동제출** (이슈 #4)
  - ⚠️ 절대시간 모드 `exam_date` 파싱 실패 시 `isNaN` 가드 (이슈 #4)
- 답안 자동 저장 (debounce 3s) + 명시적 저장 버튼
- 문제 유형별 UI
  - 객관식: 라디오
  - 단답: input
  - 서술형: textarea + 글자수 카운트
  - 작업형: 파일 업로드 / URL 입력 / 텍스트

### 감독 (Proctoring) [핵심]
- **브라우저 로컬 추론** (서버 부담 없음)
  - **얼굴 감지**: face-api.js — 미검출/다인원/시선 이탈 이벤트
  - **음성 감지**: WebAudio API RMS — 임계치 초과 시 이벤트
  - **화면 감지**: FullscreenGuard — 전체화면 이탈 이벤트
- **이벤트 배치 전송** (5초 간격, Supabase Realtime channel)
- ⚠️ **`proctoring_disabled=true` 세트 진입 시**:
  - 감독 컴포넌트 unmount
  - 상단 배너: "이 문제는 외부 도구 사용이 허용된 구간입니다 — 감독 일시 비활성화"
  - 다른 세트 이동 시 자동 재활성화 (이슈 #1)

### 제출 [핵심]
- 명시적 제출 (미답 문제 확인 dialog)
- 시간 초과 시 자동 제출
- 제출 확인 페이지

### 결과 확인 [핵심]
- 자동채점 즉시 노출 (설정에 따라)
- ⚠️ **점수 라벨 "환산 점수(100점 만점)"** (이슈 #3)
- 수동채점 완료 후 알림 (이메일/사이트 내 알림)

## Grader (채점자) 기능

### 채점 대기열 [핵심]
- 배정된 서술형·작업형 응답 목록
- 응시자 익명화 옵션 (블라인드 채점)

### 채점 인터페이스 [핵심]
- 응답 · rubric 나란히 표시
- 부분점수 슬라이더 + 코멘트
- ⚠️ **"현재 총점 {환산}/100점 (원점수 {raw}/{max})"** 표기 (이슈 #3)
- 다음 응답으로 이동 · 저장

### 크로스 채점 [확장]
- 2인 채점 · 편차 큰 응답 검토
- CrossTableDialog: 채점자간 점수 매트릭스 (모두 `/100` 표기)

## 감독 이벤트 타입 (참고)

| 이벤트 | 소스 | 심각도 |
|---|---|---|
| `face_not_detected` | face-api.js | warn |
| `multiple_faces` | face-api.js | high |
| `gaze_off_screen` | face-api.js | warn |
| `voice_detected` | WebAudio RMS | warn |
| `fullscreen_exit` | Fullscreen API | high |
| `tab_hidden` | Page Visibility API | high |
| `paste_detected` | clipboard event | info |

이벤트별 임계치·감점 규칙은 관리자 설정으로.

## MVP 범위 (M1~M4로 커버)

Admin: 문제은행 · 세트 관리(+ proctoring_disabled) · 시험 관리 · 응시 모니터링 · 채점 관리 · 결과
Applicant: 대기실 · 응시(타이머 개선) · 감독(로컬) · 제출 · 결과 확인
Grader: 채점 대기열 · 채점 인터페이스

**M5 이후 확장**: 감사 로그 · 크로스 채점 · SaaS화(멀티테넌시 · 결제 · 이메일 발송)
